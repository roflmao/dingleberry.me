It does have problems with playing more than one file in an archive and also playing files with spaces in the names. The way that I fixed these issues was to change “mplayer -fs $TMPDIR/`ls $TMPDIR`” to “find $TMPDIR -type f -exec mplayer -fs ‘{}’ +”. I also had to add “all” and “install” targets to the makefile in rarfs/doc in addition to creating the file “doc/Makefile.in” to get rarfs to install properly.
