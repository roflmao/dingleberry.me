dingleberry.me » dingleberry.me – technical blog shit http://dingleberry.me technical blog shit Fri, 06 Jun 2014 17:38:05 +0000 en-US hourly 1 http://wordpress.org/?v=3.8.1 Tweaking irssi windowlist tabs http://dingleberry.me/2009/09/tweaking-irssi-windowlist-tabs/ http://dingleberry.me/2009/09/tweaking-irssi-windowlist-tabs/#comments Fri, 04 Sep 2009 08:26:29 +0000 http://dingleberry.me/?p=438 http://anti.teamidiot.de provides a prettier windowlisting for irssi. Instead of getting a number for each window, you get the name of the channel or the name of the person you’re chatting with. This makes it easier to navigate and see where the action is going on.

This is how to do it :
mkdir -p ~/.irssi/scripts/autorun
cd ~/.irssi/scripts
wget http://anti.teamidiot.de/static/nei/*/Code/Irssi/adv_windowlist.pl
cd ~/.irssi/scripts/autorun
ln -s ../adv_windowlist.pl

in irssi:
/run autorun/adv_windowlist.pl
/set awl_display_key $Q%K|%n$H$C$S
/set awl_block -12
/statusbar window remove act
