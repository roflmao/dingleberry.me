to use this server as a http proxy (which is what we will do here), do the following as well:
aptitude install privoxy
wget http://www.doxpara.com/ozymandns_src_0.1.tgz
tar xvzf ozymandns_src_0.1.tgz
./nomde.pl -i 0.0.0.0 hesten.XXXXXXXX.no
on client, follow the same procedure to install packages and download scripts etc…

to get a shell on the dns server:
ssh -C -o ProxyCommand="./droute.pl -v sshdns.hesten.XXXXXXXX.no" user@localhost
to use the proxy we installed on the dns server:
ssh -C -o ProxyCommand="./droute.pl -v sshdns.hesten.XXXXXXXX.no" -NL 8118:localhost:8118 user@localhost
open a new window and :
export http_proxy=http://localhost:8118
and launch your fav browser..

For a full, and more thorough guide – check dnstunnel.de
